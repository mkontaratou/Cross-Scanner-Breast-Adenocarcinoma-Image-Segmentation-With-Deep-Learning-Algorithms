# Cross-Scanner-Breast-Adenocarcinoma-Image-Segmentation-With-Deep-Learning-Algorithms
Multi-scanner histopathological adenocarcinoma segmentation using foundation vision transformers (UNI2-H) with comparative decoder architectures (DeepLabV3, PixelFPN, ASPP, Lite Query). Includes full training, post-processing, and evaluation pipelines for COSAS 2024.
